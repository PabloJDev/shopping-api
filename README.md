# 🛒 Shopping API

API simples de lista de compras desenvolvida com NestJS.  
Permite adicionar e listar itens de forma rápida e prática.

---

## 🚀 Funcionalidades

- ✅ Adicionar item à lista de compras
- ✅ Listar todos os itens cadastrados

---

## 🧪 Tecnologias utilizadas

- Node.js
- NestJS
- TypeScript

---

```bash
git clone https://github.com/seu-usuario/shopping-api.git
cd shopping-api
